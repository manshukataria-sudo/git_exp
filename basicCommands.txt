Making any folder a git repository
Use the command => git init


Firstly all files are untracked to make them staged use the command
we verify the changes in the staged area before commiting carefully
git add <path>


Use the command 
git status
we can also use the command => git status -s , for more precision
to check if the files are untracked or in the staged area



To make the file untracked use the command
git rm --cached <path>


To see the commit history use the command
git log
