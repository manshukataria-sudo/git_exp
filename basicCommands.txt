Making any folder a git repository
Use the command => git init


Firstly all files are untracked to make them staged use the command
we verify the changes in the staged area before commiting carefully
git add <path>


Use the command 
git status
we can also use the command => git status -s , for more precision
to check if the files are untracked or in the staged area



To make the file untracked use the command
git rm --cached <path>


To see the commit history use the command
git log


To see all the commits from all branches use the command
git log --all
and in one line use the command
git log --all --oneline




Whenever a new branch is created, all the commits of the branch from which a new branch is created 
are added to the new branch also.


To see the graph of commits use the command
git log --graph


To delete a git branch use the command, to delete branch should not be checked out
means we should be on anyother branch
git branch --delete <branchName>
